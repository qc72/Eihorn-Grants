# Grant Collaboration Network — simple stable version

This version deliberately returns to the original application structure.

## Stability choices

- No `st.fragment`
- No automatic reruns
- No animated-layout control
- No background update checker
- One five-minute CSV cache
- Manual **Reload CSV now** button
- Last-good and bundled-CSV fallback
- Original detailed node and edge grant panel

## Streamlit Secrets

```toml
CSV_SOURCE = "YOUR_GOOGLE_SHEETS_PUBLISHED_CSV_URL"
```

Use the URL generated by Google Sheets **File → Share → Publish to web → CSV**.

## Deployment

Upload these files to the repository root:

- `app.py`
- `grant_network.py`
- `requirements.txt`
- `Grants-Network.csv`

Use `app.py` as the Streamlit entrypoint. Python 3.12 is recommended.
